IDVault
A Secure Offline ID Card Manager for Android
📌 Overview

IDVault is a privacy-focused Android application that allows users to securely store and manage their important ID cards (Aadhar, PAN, College ID, Driving License, etc.) directly on their device.

Unlike cloud-based solutions, IDVault works 100% offline, ensuring that your sensitive data never leaves your phone.

🔐 Key Features
Security
🔐 PIN Lock – Protect your vault with a custom 4-digit PIN
📴 Fully Offline – No internet, no cloud, no external servers
Card Management
🗂️ Card Storage – Store multiple ID card images securely
✏️ Rename Cards – Easily update card names anytime
🗑️ Delete Options – Swipe to delete or use long-press menu
Usability
🔍 Search Functionality – Quickly find cards by name
📤 Share Feature – Share card images directly from the app
👤 Profile Section – View stats, change PIN, manage vault
UI/UX
🌙 Dark Theme – Clean, premium vault-style interface
📱 Smooth Navigation – Fragment-based bottom navigation
📸 Screens (App Flow)
PIN Screen – Secure login access
Home Screen – Grid view of stored ID cards
Search Screen – Filter and find cards quickly
Profile Screen – Stats, settings, and PIN management
🛠️ Tech Stack
Language: Java
UI: XML Layouts
Storage: Internal File Storage + SharedPreferences
Navigation: Fragment-based Bottom Navigation
Image Sharing: FileProvider (AndroidX)
Min SDK: 24 (Android 7.0)
Target SDK: 36
🎯 Core Idea

IDVault is built on a simple principle:

Your personal data should stay personal.

By eliminating cloud dependency, the app ensures:

Better privacy
Faster access
No risk of data leaks
